AgerBot Instruções para Avaliação

AgerBot

O AgerBot é um chatbot. Um assistente eletrônico, na plataforma Telegram para o usuário interagir com os dados coletados pelo robô agrícola AGER.

O AGER é um robô movido a energia solar, capaz de auxiliar na jardinagem e produção agrícola. Ele é planejado tanto para funcionar em estufas ou ao ar livre e se propõe a ter um funcionamento simples e intuitivo para o seu usuário.

Roteiro de Teste

Para realizar este teste de usuabilidade você deve seguir os passos abaixo:

  1. Ter instalado o app Telegram, disponível para diversas plataformas (iOS, Android, Windows Phone, Windows, Mac OS, e Linux);
  2. Buscar pelo contato agerbot ou @ager_telegram_bot;
  3. Registrar um AGER utilizando um dos cinco números de série abaixo (pode escolher qualquer um deles);
  4. Criar uma nova rota de trabalho para o AGER;
  5. Verificar os últimos dados coletados;
  6. (Opcional) Utilizar livremente o app.
  7. Responder o questionário do link: https://goo.gl/forms/4tuw8P8a2UinMweC2

Números de Série
